# Quality Guard
A simple Python project for tracking quality standards and metrics.

## Usage
1. Create a `QualityGuard` instance.
2. Add quality standards using `add_quality_standard`.
3. Add quality metrics using `add_quality_metric`.
4. Visualize quality trends using `visualize_quality_trends`.
5. Persist quality standards using `persist_quality_standards`.

## Testing
Run `pytest` to execute the tests.
